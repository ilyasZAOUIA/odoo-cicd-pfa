====================================
Gestion Matériel Construction
====================================

Module de démonstration développé dans le cadre du projet PFA
« Pipeline CI/CD pour le déploiement de modules Odoo » chez BAAM.

Fonctionnalités
===============

* Gestion du matériel de construction (nom, référence, quantité, prix)
* Calcul automatique de la valeur du stock
* Contrainte empêchant une quantité négative
